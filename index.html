soundcloud2000

SoundCloud without the stupid CSS files

Download as .zip Download as .tar.gz View on GitHub

soundcloud2000 is terminal client for the SoundCloud platform. Currently runs only on OSX using CoreAudio and NCurses.

This hack was built during the Music Hack Day Stockholm 2013.

For installation instructions, feature list and todos, please checkout the README.
